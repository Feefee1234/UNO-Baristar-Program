การตั้งค่าครั้งแรก: เชื่อมต่อแอปนี้กับ Google Sheet เพื่อให้คะแนนอัปเดตอัตโนมัติ

1. เปิด Google Sheet → ไฟล์ → แชร์ → เผยแพร่เป็นเว็บ
2. เลือกแท็บชีตที่ถูกต้อง → เลือกรูปแบบ CSV
3. คลิก เผยแพร่ แล้วคัดลอกลิงก์
4. วางด้านล่างแล้วกด เชื่อมต่อ

Sheet columns must be in this order:
A: Store   B: Store Code   C: Name   D: ID   E: Nickname   F: Position   G: Start Date   H: Resigned   I: Total Points   J: Tier
